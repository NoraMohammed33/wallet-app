                                              Wallet App
                                                    
A simple React application for managing a user's wallet with gamification features.

Features

Show the user's current balance.
Allow users to add funds to their wallet by entering an amount and clicking a button.
Allow users to withdraw funds from their wallet by entering an amount and clicking a button.
The withdraw button disabled if the user's balance is insufficient.
Reward users with bonus funds when they reach certain milestones:
Add $100 to the wallet: reward $5 bonus funds.
Add $500 to the wallet: reward $20 bonus funds.
Add $1000 to the wallet: reward $50 bonus funds.
Bonus funds will be added automatically when a milestone is reached.

Installation
1-Clone the repository:
   git clone  https://github.com/NoraMohammed33/wallet-app.git
   cd wallet-app

2-Install dependencies:
   you can run:  `npm install`

Usage
Start the development server:
   you can run: `npm start`

Open http://localhost:3000 to view it in your browser.

The page will reload when you make changes.
